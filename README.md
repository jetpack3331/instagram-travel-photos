# Travel App

Show on map your photos from your trip(s)

## Contents

- Installation
- Setup
- Testing

### Installation

To install application run
`npm install` or `yarn install`\

### Setup

To run local development run
`npm run start`
It will run app on http://localhost:8080

### Testing

To run test run
`npm run test`

to run tests alway on file changes run
`npm run test:watch`
